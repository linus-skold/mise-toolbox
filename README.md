Just a little toolbox of stuff that i use for [mise](https://mise.jdx.dev/)

The mise.toml should be placed in the global mise config folder usually found under the users home folder
~/.config/mise



## Powershell
install the spinner with
```pwsh
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/linus-skold/mise-toolbox/main/powershell/spinner.ps1" -OutFile "$env:USERPROFILE\\spinner.ps1"
```
